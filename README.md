# Ollama Chat

Tinkering again. This time with LLMs and trying to create a chat bot client. At the moment I'm assumed there's a local instance of [Ollama](https://ollama.com/) running (unlikely on a mobile, but then again with termux and a light model it could be possible). The intention is to point the client to a known IP on your local network.

I'm following https://docs.ollama.com/api/introduction for the implementation. Yes, I know there's a [JS library](https://github.com/ollama/ollama-js) but its fun to see how far you can get on REST alone.
