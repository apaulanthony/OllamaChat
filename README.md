# Ollama Chat

Tinkering again. This time with LLMs and trying to create a chat bot client. At the moment I'm assumed there's a local instance of [Ollama](https://ollama.com/) or [LM Studio](https://lmstudio.ai/) for the client to connect to. A mobile would be an unlikely target but then again, with termux and a light model it could be possible, or there's https://github.com/google-ai-edge/gallery.

I'm following https://docs.ollama.com/api/introduction and https://lmstudio.ai/docs/developer/rest for the implementation. Yes, I know both [Ollama](https://github.com/ollama/ollama-js) and [LM Studio](https://github.com/lmstudio-ai/lmstudio-js#readme) have npm libraries but it's fun to see how far you can get on REST alone.
